# Lesson1 - Boot Process

## Objective
Understand what happens from the moment a computer powers on until the operating system loads. 

## Boot Sequence

1. Power Button
2. Power Supply
3. CPU
4. BIOS/UEFI
5. POST
6. Boot Loader
7. Kernel
8. Drivers
9. Login Screen

## What I Learned

Today I learned that the operating system is not the first thing that starts when a computer powers on.
The BIOS/UEFI checks the hardware before the operating system loads.
The boot loader is responsible for finding the operating system.
The kernel is the core of the operating system and communicates with the hardware.

## Why This Matters

Understanding the boot process will help me troubleshoot computers that fail during startup.
