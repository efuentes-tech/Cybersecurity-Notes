# Lesson 4 - Storage

## Objective
Understanding of the hardware path. Understanding how a program moves from permanent storage to execution. 

## What I learned
Today I learned what is storage. Storage is long term memory. Storage remembers everything, even when the power is turned off. Two main types of storage are Hard Disk Drive(HDD) and Solid State Drive (SSD). HDD has moving parts and SSD is digital. HDD can be more prone to damage which is why many computers now a days use SSD. Storage also has a file system so the operating system can easily find your files. File system is the organization system for storage that keeps track of file names, locations, permissions, free space and much more. Storage have capacity and speed. Capacity tells you how much you can store and spped tells you how quicly data can be accessed. 

## Interview Question
Interview Question:
“What is the purpose of storage in a computer?”
Storage provides long-term, non-volatile memory for a computer. It permanently stores the operating system, applications, files, and user data even when the computer is powered off. When a program is opened, it is copied from storage into RAM so the CPU can execute it. 

## Cybersecurity Connection
Why should a cybersecurity professional understand storage?
Several reasons:

Digital forensics often involves analyzing data stored on drives.
Malware can hide in files or modify the boot process on storage devices.
Ransomware encrypts files stored on disks.
Secure deletion matters because deleting a file doesn't always remove the underlying data immediately.

Understanding storage helps you understand both system performance and many types of security incidents.
