# Lesson 3 - RAM 

## Objective
Understand what RAM actually is, why computers need RAM, and how it is different from storage. 

## What I learned
Today I learned that RAM stands for Random Access Memory.  RAM is basically a temporary workspace but exists because the CPU is incredibly fast but the SSD is much slower. If the CPU had to wait on the SSD in every interaction then your computer would feel painfully slow. RAM is in between to act like a fast workspace. I also learned that when RAM fills up, your computer can become painfully slow so the operating system would start moving data from RAM back to the SSD. Knowing this information, you can answer the question if RAM makes a computer faster. The answer is no, but if you have one computer which is using 90% or more of RAM, then upgrading your RAM would speed things up but if another computer is using 30% of RAM, then more RAM would not speed up your computer. 

## Real-World Example
Customer says: "My computer is slow."
I check task manager and check the Memory usage. If RAM is constantly above 90%, we might need an upgrade. If RAM is onlyh at 30%, then RAM probably is not the issue. 

## Troubleshooting Scenario
Investigate and gather evidence. 
Questions I can ask myself.
How much RAM is installed? 
How much RAM is being used? 
Which application is using RAM?
Has usage increased recently?
Do we have enough RAM for this workload?

## Interview Question:
"What is RAM, and why is it important?"
RAM is the computer's temporary working memory. When programs are opened, they are copied from strorage into RAM because RAM is much faster than an SSD. The CPU reads instructions from RAM while programs are running. If the system runs out of RAM, the operating system starts moving data back to storage, which can significantly slow down performance. 

## Key Takeaways
Storage stores.
RAM holds.
CPU works.
