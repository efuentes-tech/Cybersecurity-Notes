# Lesson 5 - The Operating System

## Objective 
Understand what and operating system really is and how the operating system brings everything together such as the CPU, RAM, and storage. 

## What I Learned
Today, I learned that the operating system acts as a manager to a computer. It coordinates everything. I learned that the responsibilities of the OS is process management, memory management, file management, device management, and security. For example, the OS decides which program gets CPU time first, decides where applications live in RAM, Keeps track of files, knows how to communicate with devices, and can decide who can and can not open files. The OS coordinates and organizes everything. I also learned that different operating systems all solve the same problems but the difference is how they do it. Different styles but the same responsibilities. 

## Real World Example:
“My printer isn’t working.”

We are troubleshooting the operating system, not the printer.

## Interview Question:
“What is an operating system?”
An Operating System is software that manages a computer's hardware and software resources. It acts as the intermediary between the user, applications, and hardware by managing processes, memory, storage, devices, files, and security. Without an operating system, applications would not be able to communicate with the hardware efficiently.

## Cybersecurity Connection

Security professionals spend a lot of time understanding operating systems because:

Malware often targets the operating system.
User permissions are enforced by the operating system.
Logs come from the operating system.
Security patches update the operating system.
Access control is managed by the operating system.

If you don't understand the operating system, you'll struggle to understand how attacks happen or how to defend against them.

## Key Takeaway
The operating system is the manager. Storage stores. RAM holds. CPU works. The operating system manages.
